Linux安装python3

发表于 2019-08-10 | 更新于 2019-08-11 | 分类于 Memo | | 阅读次数：
本文字数： 2.8k | 阅读时长 ≈ 3 分钟

环境

CentOS 6

linux原本就安装了Python2，而且这个Python2不能被删除，因为有很多系统命令，比如yum都要用到。

输入命令Python，查看得知是Python2.6版本

下面介绍如何在保留原python2的情况下，安装python3

安装python3

下载依赖包

yum -y install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gcc make

新建下载目录

我们将下载目录设置为/usr/local/python3

mkdir /usr/local/python3

进入下载目录

cd /usr/local/python3

下载python3 安装包

从python官网可以找到下载链接，我选择的是python3.6.0版本

wget https://www.python.org/ftp/python/3.6.0/Python-3.6.0.tgz

解压

tar -xvzf Python-3.6.0.tgz

进入目录

cd Python-3.6.0

添加配置

执行以下命令，其中–prefix用于设置Python的安装路径，我们将/usr/python设为安装路径

./configure --prefix=/usr/python

编译源码

make

执行安装

make install

为Python3和pip创建软链接

创建软链接的命令是ln -s a b ， a就是源文件，b是链接文件名，其作用是当进入b目录，实际上是链接进入了a目录

不创建软链接的话每次执行要进入到Python3.6的安装目录

注意：此处的源文件路径按照python安装路径进行修改

ln -s /usr/python/bin/python3 /usr/bin/python3

ln -s /usr/python/bin/pip3 /usr/bin/pip3

配置环境变量

vi ~/.bash_profile

使配置生效

source ~/.bash_profile

验证是否装成

which python3

python3 -V

使用方式

我们刚才设置了软链接，可以直接使用下方的命令来调用python3和pip

python3 ...

pip3 ...
